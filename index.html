🚀 Quick Start - Copy & Paste

1. Reset an episode:
curl -X POST https://kiransin-openaudit.hf.space/reset?task_id=model_card_easy
2. Submit a finding:
curl -X POST https://kiransin-openaudit.hf.space/step \ -H "Content-Type: application/json" \ -d '{"pillar":"model_card","finding_type":"missing_field","target_field":"license","description":"Missing license","severity":2}'
3. Check episode state:
curl -X GET https://kiransin-openaudit.hf.space/state
4. List all 13 tasks:
curl -X GET https://kiransin-openaudit.hf.space/tasks
5. Health check:
curl -X GET https://kiransin-openaudit.hf.space/health
6. Run the full AI agent (requires OpenAI key):
export API_BASE_URL="https://kiransin-openaudit.hf.space" export MODEL_NAME="gpt-3.5-turbo" export OPENAI_API_KEY="your_openai_key" python inference.py
