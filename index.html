API Reference
POST /reset?task_id={task_id} Start episode
POST /step Submit finding
GET /state Episode state
GET /tasks List all 13 tasks
GET /health Health check
GET /docs Swagger UI
Quick Start
# 1. Health check curl https://kiransin-openaudit.hf.space/health # 2. List all 13 tasks curl https://kiransin-openaudit.hf.space/tasks # 3. Reset an episode curl -X POST https://kiransin-openaudit.hf.space/reset\ ?task_id=model_card_easy # 4. Submit a finding curl -X POST https://kiransin-openaudit.hf.space/step \ -H "Content-Type: application/json" \ -d '{"pillar":"model_card", "finding_type":"missing_field", "target_field":"license", "description":"Missing license field", "severity":2}' # 5. Check episode state curl https://kiransin-openaudit.hf.space/state # 6. Python — run the baseline agent pip install openai requests export HF_TOKEN="your_hf_token" export API_BASE_URL="https://router.huggingface.co/v1" export MODEL_NAME="Qwen/Qwen2.5-72B-Instruct" export ENV_API_URL="https://kiransin-openaudit.hf.space" python inference.py # 7. Python — single step example import requests obs = requests.post("https://kiransin-openaudit.hf.space/reset", params={"task_id": "dataset_qc_easy"}) result = requests.post("https://kiransin-openaudit.hf.space/step", json={"pillar": "dataset_qc", "finding_type": "null_values", "target_field": "columns", "description": "Null values found", "severity": 2}) print(result.json()["reward"])
