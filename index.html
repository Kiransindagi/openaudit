📡 API Endpoints

POST /reset?task_id={task_id} Start a new audit episode
POST /step Submit an audit finding
GET /state Get current episode state
GET /tasks List all 12 auditing tasks
GET /health Service health check
GET /docs Interactive Swagger documentation
🚀

Quick Start

curl -X POST /reset?task_id=model_card_easy
curl -X POST /step -d '{"pillar":"model_card",...}'

Run the Agent

python inference.py
