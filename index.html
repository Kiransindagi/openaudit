📡 API Endpoints

POST /reset?task_id={task_id} - Start a new audit episode
POST /step - Submit an audit finding
GET /state - Get current episode state
GET /tasks - List all 13 tasks
GET /health - Health check
GET /docs - Swagger UI

🚀 Quick Start

Reset:
curl -X POST https://kiransin-openaudit.hf.space/reset?task_id=model_card_easy
Step:
curl -X POST https://kiransin-openaudit.hf.space/step -H "Content-Type: application/json" -d '{"pillar":"model_card","finding_type":"missing_field","target_field":"license","description":"Missing license","severity":2}'
Tasks:
curl -X GET https://kiransin-openaudit.hf.space/tasks
